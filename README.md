# Training Questions

## Anagram
In this question, anagram checks whether the two given strings are the same and contain the same characters. If all characters match correctly, then it returns `true`.

## Deep Merge
In this question, we merge two objects into a single object.

## Remove Duplicate
This function removes duplicate elements from the array and returns a new array with unique elements.

## Nested Array Sum
In this question, we have to find the sum of elements present inside nested arrays.

## Missing Number
In this question, we have to find the missing number from the given array.

## Subarray with Given Sum
In this question, we have to check which elements of the array produce the given sum value.

## Object Update
In this question, we have to update the value of an object.

## Group By Property
In this question, we have to group objects based on the given property.

## String Without Repeating
In this question, we have to find a string after removing duplicate characters.

## Group Anagrams
In this question, we have to group words that contain the same characters.


# User Defined Methods

## Count Words
This method counts how many words are present in the string.

## Every
This method checks whether all conditions are true. If some conditions are true and some are false, then it returns `false`.

## Find Duplicate
In this question, we find duplicate values in an array.

## Palindrome
This question checks whether the given string is a palindrome or not. A palindrome means the string is the same when reversed.

## At
This function returns the element at the given index in the array.

## Concat
This method combines two arrays and returns a new array.

## Include
In this question, we check whether the given character is present in the string or not.

## IndexOf
This method finds the index of the given character in the string and returns its index.

## Join
This function joins the elements of an array and returns a string.

## Pop
This method removes the last element from the array and returns the new array without the last element.

## Push
This method adds a new element at the end of the array and returns the updated array.

## Remove Duplicate
This function removes duplicate elements from the array and returns a new array with unique elements.

## Replace
This function replaces the given value with a new value in the string and returns the updated string.

## Reverse
This function reverses the given string and returns the new reversed string.

## Shift
This method removes the first element of the array.

## Slice
This function returns elements from the given start index to the end index in the array.

## Sort
This method arranges array elements in ascending or descending order.

## Split
This function splits the given string into an array of words and returns the new array.

## Trim
This function removes spaces from the beginning and end of the string and returns the new string.

## Unshift
This method adds a value at the first index of the array.

## Some
This method checks whether some conditions are true. If at least one condition is true, then it returns `true`.

## ToString
This function returns a string representation of the array elements separated by commas.
